(Q1-Data Type 1) 25 Point

Sebuah sistem keamanan digital menggunakan kode rahasia berupa bilangan 4 digit. Untuk menjaga keamanan data, sistem tidak menyimpan kode tersebut secara langsung, tetapi mengubahnya menjadi nilai sandi (encryption code) menggunakan rumus berikut.

A digital security system uses a secret code consisting of a 4-digit number. To maintain data security, the system does not store the code directly; instead, it converts it into an encryption value (k) using the following formula:

$$k = (d_1 + d_2) \times (d_3 + d_4) + (p - q)$$

Nilai $d_i$ adalah digit ke-i dari kode rahasia yang diberikan (baca digit dari kiri ke kanan). Nilai p adalah hasil penjumlahan digit pertama dan terakhir, sedangkan nilai q adalah hasil pengurangan digit kedua oleh digit ketiga.

Here: $d_i$ is the i-th digit of the secret code (read from left to right).

p is the sum of the first and last digits.

q is the result of subtracting the third digit from the second digit.

Masukan berupa kode rahasia yang dinyatakan oleh sebuah bilangan bulat m ($1000 \le m \le 9999$) berdigit empat.

Keluaran berupa nilai sandi k hasil enkripsi yang dilakukan.

Input: A secret code represented by a four-digit integer m ($1000 \le m \le 9999$).

Output: The encryption value k produced by the system.

Answers

Plaintext



program Enkripsi

kamus

    m, d1, d2, d3, d4, p, q : integer

    k : integer

algoritma

    input(m)

    // 1. cari nilai d1 s.d. d4

    d1 = m div 1000

    d2 = m div 100 mod 10

    d3 = m div 10 mod 10

    d4 = m mod 10

    // 2. Hitung nilai p dan q

    p = d1 + d4

    q = d2 - d3

    // 3. hitung nilai k

    k = (d1 + d2) * (d3 + d4) + (p - q)

    output(k)

endprogram

(Q2-Data Type 2) 25 Point

Sebuah Cake berbentuk lingkaran dengan tebal tertentu. Cake tersebut akan dipotong menjadi N bagian sama besar. Bagian berbentuk seperti segitiga dengan bagian runcing mengarah ke arah pusat cake. Apabila cake tersebut diasumsikan memiliki bentuk menyerupai tabung, maka buatlah program untuk menghitung luas permukaan satu potongan cake saja (Asumsi satuan panjang dalam centimeter/cm dan nilai $\pi = 3.14$).

A cake has a circular shape with a certain thickness. The cake will be cut into N equal pieces. Each piece is shaped like a triangle, with its pointed tip directed toward the center of the cake. If the cake is assumed to have the shape of a cylinder, create a program to calculate the surface area of one piece of the cake (assuming all measurements are in centimeters).

Masukan berupa tiga bilangan yang menyatakan besarnya nilai N (banyaknya potongan cake) dan diameter (d) dan tebal/tinggi (t) dari cake.

Keluaran berupa luas permukaan satu buah potongan cake (seperti gambar di samping).

Input: Three numbers representing: $N \rightarrow$ the number of cake slices, d the diameter of the cake, and $t \rightarrow$ the thickness (height) of the cake.

Output: The surface area of one slice of the cake, as shown in the accompanying illustration.

Answers

Plaintext



program Cake

kamus

    N : integer

    d, t, r : real

    keliling, tepi, luasAlas, luasTepi : real

    luasLuar, luasPotongan : real

algoritma

    input(N, d, t)

    r = d / 2

    // 1. Hitung keliling & luas lingkaran

    keliling = 2 * 3.14 * r

    luas = 3.14 * (r * r)

    // 2. Hitung luas kulit tepi dalam, alas/atas, & luar

    luasTepi = r * t

    luasAlas = luas / N

    luasLuar = (keliling / N) * t

    // 3. Hitung luas potongan cake, 2 tepi dalam + 2 alas/atas + 1 luar

    luasPotongan = 2 * luasTepi + 2 * luasAlas + luasLuar

    output(luasPotongan, "cm2")

endprogram

(Q 3-Looping) 25 Point

Sebuah program digunakan oleh Posyandu untuk menghitung Body Mass Indeks (BMI) dan Health Indicators (I) dari bayi-bayi yang melakukan imunisasi. Data yang diproses adalah data berat badan (w), tinggi badan (h) dan lingkar kepala bayi (c). Selanjutnya perhitungan dapat menggunakan rumus:

A program is used by the Posyandu (community health center) to calculate the Body Mass Index (BMI) and Health Indicator (I) of infants who receive immunizations. The data processed include the baby's weight (w, in grams), height (h, in centimeters), and head circumference (c, in centimeters).

$$bmi = \frac{10w}{(h)^2}$$

dan

$$I = \frac{bmi + (c/10)}{2}$$

Masukan terdiri dari $N+1$ baris. Baris pertama adalah bilangan bulat N yang menyatakan banyaknya bayi. N baris berikutnya masing-masing adalah berat badan (w), tinggi badan (h) dan lingkar kepala bayi (c).

Keluaran terdiri dari $N+1$ baris, N baris pertama adalah nilai BMI dan I dari setiap bayi. Baris terakhir adalah rata-rata BMI dan I dari semua bayi.

Input: The input consists of $N+1$ lines. The first line contains an integer N, representing the number of babies. The next N lines each contain three numbers: the baby's weight (w), height (h), and head circumference (c).

Output: The output also consists of $N+1$ lines. The first N lines show the BMI and I values for each baby. The final line shows the average BMI and average I for all babies.

Jawaban

Plaintext



program Posyandu

    N, j : integer

    BMI, HI, w, h, c, total_BMI, total_HI : real

algoritma

    input(N)

    total_BMI = 0

    total_HI = 0

    for j = 1 to N do

        input(w, h, c)

        BMI = w / (h * h) * 10

        HI = (BMI + (c / 10)) / 2

        total_BMI = total_BMI + BMI

        total_HI = total_HI + HI

        output(BMI, HI)

    endfor

    output(total_BMI / N, total_HI / N)

endprogram
