# AFS RAG Dataset

This directory contains the Knowledge Base for the Affective Feedback System (AFS) RAG implementation.

## Content
- **Pedagogical Strategies**: Documents describing how to handle different student situations.
- **Psychological Guidelines**: Reference materials on student motivation and psychology.
- **Phrasing Templates**: Examples of encouraging or constructive language.

## Purpose
To provide the retrieval base for generating context-aware affective feedback.
