# AFS Multi-Agent Dataset

This directory contains datasets for the Affective Feedback System (AFS).

## Content
- **Interaction Logs**: History of student-tutor interactions.
- **Affective States**: Labels or indicators of student frustration, confusion, engagement, etc.
- **Feedback Examples**: Database of appropriate responses for various affective states.

## Purpose
To train and test the agents responsible for providing emotional and pedagogical support.
