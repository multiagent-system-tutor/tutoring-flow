# OCR + LLM Dataset

This directory contains the dataset for the OCR + LLM pipeline.

## Structure
- Store input images containing text (handwritten or printed).
- Store expected outputs which may include:
    - Raw extracted text.
    - Contextually corrected text (by LLM).
    - Structured extracted data.

## Purpose
Used to evaluate the performance of combining OCR with LLM for context correction and formatting.
